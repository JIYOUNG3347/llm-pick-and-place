박스를 집는
상태머신

실행 루프는 매 틱 status → act → env.step → perceive를 돈다. 그래서 모든 상태의 IN은 WorldState(큐브 GT pose + 로봇 상태), OUT은 Action (ee_target + gripper)이다. 툴이 아래를 향하므로 TCP_z = wrist_z − tcp_offset, 손끝이 큐브 중심에 닿게 목표를 잡는다.

grasp_wrist_z = cube_z + tcp_offset_z(0.107)  → TCP_z = cube_z (큐브 중심)
TCP_z = wrist_z − tcp_offset_z  → 손끝 중점

상수 (현재값)

tcp_offset_z 0.107 obj_half_h 0.021 DESCEND_RATE 0.003/step DESCEND_TOL 0.005 DESCEND_MAX 400 TOL_COARSE 0.05 min_grasp_finger 0.010 grasp_dist_thresh 0.045 PICK_TIMEOUT 1600 gripper OPEN/CLOSE 0.0/1.0
팔 이동 그리퍼 동작 성공
APPROACH
큐브 위 0.40m로 이동, 그리퍼는 열어둠
IN cube(ox,oy,oz) · wrist OUT ee_target=(ox, oy, 0.40) · gripper=OPEN
ee_dist([ox,oy,0.40]) < 0.05  →  grasp_wrist_z = oz + 0.107 확정
DESCEND
cmd_z를 3mm/step씩 grasp_wrist_z까지 내림 → TCP를 큐브 중심으로
IN cube(ox,oy,oz) · wrist_z OUT cmd_z=max(grasp_wrist_z, cmd_z−0.003); ee_target=(ox,oy,cmd_z) · gripper=OPEN
|wrist_z − grasp_wrist_z| < 0.005  OR  wait ≥ 400
CLOSE
위치 고정한 채 그리퍼 닫고 50틱 대기 (관절 이동 시간 확보)
IN cube(ox,oy,oz) · wait OUT ee_target=(ox, oy, grasp_wrist_z) · gripper=CLOSE
wait ≥ 50
WAIT_GRASP
IsaacEnv의 FixedJoint 파지 검출 결과를 폴링
IN world.robot.holding OUT = CLOSE와 동일 (gripper=CLOSE 유지)
실패: wait ≥ 60 → FAILURE

파지 검출 — FixedJoint 바인딩 (4개 AND)

  1. gripper_cmd > 0.5  (CLOSE=1.0)
  2. holding is None  (아직 안 잡음)
  3. closed_amount ≥ 0.010  panda: 0.04−finger_val ≥ 0.010 → finger_val ≤ 0.030
  4. dist < 0.045  ‖TCP − 큐브‖ (3D, world)
성립 → holding=cube, USD FixedJoint 생성(kinematic)  ·  해제: gripper_cmd ≤ 0.5 → FixedJoint 제거
holding ≠ None
LIFT
0.40m로 들어올림 (FixedJoint로 큐브가 동반)
IN cube[:2] (들고 있으면 따라옴) · wrist OUT ee_target=(ox2, oy2, 0.40) · gripper=CLOSE
ee_dist([ox2,oy2,0.40]) < 0.05
✓ SUCCESS
전역 가드 — 모든 상태에서 steps ≥ 1600 (PICK_TIMEOUT) → FAILURE
PlaceSkill (요약)
place_z = tz + 2·obj_half_h + 0.015 + tcp_offset_z — 들고 있는 큐브 바닥이 타깃 상단 +15mm
HOVEROUT (tx,ty,0.40) · CLOSE → ee_dist < 0.05
DESCENDOUT (tx,ty,place_z) · CLOSE → ee_dist < 0.010 (FINE)
OPENOUT (tx,ty,place_z) · OPEN → wait ≥ 20
WAIT_RELEASEOPEN 유지 → holding=None OR wait ≥ 30  (FixedJoint 제거 감지)
RETREATOUT (tx,ty,0.40) · OPEN → ee_dist < 0.05 → SUCCESS
APPROACH → DESCEND → CLOSE → WAIT_GRASP → LIFT · grasp = 4-AND FixedJoint
